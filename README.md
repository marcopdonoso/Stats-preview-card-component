### What I learned

What i've learned whit this challenge is to apply a colored box in front of a image, to simulate an filter effect.

```html
<div class="purple-filter"></div>
```
```css
.purple-filter {
    background-color: rgba(87, 4, 139, 0.65);
    position: absolute;
    right: 0;
    width: 540px;
    height: 446px;
    z-index: 1;
}
```
